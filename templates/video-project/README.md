# vstack Video Project Template

Use this folder structure for one video project.

Duplicate this template for each new video.

## Folder structure

```text
video-project/
  00_brief/
    greenlight.md
  01_script/
    script.md
  02_preproduction/
    shot-list.md
    camera-lighting-plan.md
  03_production/
    shoot-log.md
    footage-notes.md
  04_post/
    edit-notes.md
  05_publish/
    thumbnail-concepts.md
    distribution-package.md
  06_review/
    post-mortem.md
```

## Workflow

1. Start with `00_brief/greenlight.md`.
2. Do not write the script until the concept has been greenlit.
3. Do not shoot until the shot list and camera-lighting plan are usable.
4. Do not publish until the thumbnail and distribution package are complete.
5. After publication, complete the post-mortem before starting the next related video.

## Naming convention

Recommended project folder format:

```text
YYYY-MM-topic-slug
```

Example:

```text
2026-04-lexicon-benign
```

## Pilot rule

For the first pilot, keep the video short.

Target runtime: 60–180 seconds.
